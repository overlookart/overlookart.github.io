## 领导职务层次

中国公务员领导职务分为10个层次：

## 职务与职级对应关系

担任领导职务的公务员，其领导职务与职级存在对应关系：

| 领导职务 | 对应最低职级 | 对应级别范围 |
|---------|------------|------------|
| 厅局级正职 | 一级巡视员 | 13-8级 |
| 厅局级副职 | 二级巡视员 | 15-10级 |
| 县处级正职 | 二级调研员 | 18-12级 |
| 县处级副职 | 四级调研员 | 20-14级 |
| 乡科级正职 | 二级主任科员 | 22-16级 |
| 乡科级副职 | 四级主任科员 | 24-18级 |


## 不同级别城市的行政级别

中国城市按行政级别分为四类，其领导职务级别相应不同：

| 城市类型 | 代表城市 | 市长级别 | 副市长级别 |
|---------|---------|---------|-----------|
| 直辖市 | 北京、上海、天津、重庆 | 省部级正职 | 省部级副职 |
| 副省级城市 | 深圳、广州、杭州、成都等 | 副省级（省部级副职） | 厅局级正职 |
| 地级市 | 各省地级市 | 厅局级正职 | 厅局级副职 |
| 县级市 | 各省县级市 | 县处级正职 | 县处级副职 |


## 晋升条件

公务员晋升职级需满足以下最低任职年限条件：

| 晋升职级 | 最低任职年限 |
|---------|------------|
| 二级巡视员 | 任一级调研员4年以上 |
| 一级调研员 | 任县处级正职或二级调研员3年以上 |
| 二级调研员 | 任三级调研员2年以上 |
| 三级调研员 | 任县处级副职或四级调研员2年以上 |
| 四级调研员 | 任一级主任科员2年以上 |
| 一级主任科员 | 任乡科级正职或二级主任科员2年以上 |
| 二级主任科员 | 任三级主任科员2年以上 |
| 三级主任科员 | 任乡科级副职或四级主任科员2年以上 |
| 四级主任科员 | 任一级科员2年以上 |
| 一级科员 | 任二级科员2年以上 |
